# Jogo-da-Velha
Este Jogo da Velha foi criado por Jonas Invernizzi e Felipe Trevisan, durante o primeiro ano do Ensino Médio Técnico em Informática, utilizando HTML, CSS e JavaScript.

O jogo mantém a clássica mecânica de alinhar três símbolos, X ou O, oferecendo uma experiência simples, estratégica e envolvente, perfeita para partidas rápidas ou desafios entre amigos.

A aplicação combina design limpo e intuitivo com uma lógica robusta, garantindo jogabilidade dinâmica e responsiva. Este projeto reflete habilidades em desenvolvimento web, planejamento e trabalho em equipe, demonstrando a aplicação prática do conhecimento técnico em um projeto funcional e educativo.
